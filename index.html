01
// POINT D'ENTRÉE — OBJECTIF PRIMAIRE
ACCÈS À LA BOUTIQUE
Reconstituer les paniers d'achat des 4 clients sur la borne d'entrée pour déverrouiller l'accès.
⚡ REQUIS — POINT DE DÉPART
⚠ ATTENTION CRITIQUE Ne pas oublier d'inclure les partenaires dans la reconstitution des paniers. Erreur fréquente des équipes.
PROCÉDURE D'ACCÈS
  • 01.Localiser la borne d'entrée à l'extérieur de la boutique.
  • 02.Identifier les 4 clients — chacun a effectué des achats différents. Récupérer les informations sur ce qu'ils ont acheté.
  • 03.Reproduire exactement le contenu de chaque panier sur la borne. L'ordre et les quantités sont importants.
  • 04.Vérifier que les partenaires associés aux clients sont bien inclus dans la sélection — c'est l'élément le plus souvent oublié.
  • 05.Valider sur la borne une fois les 4 paniers correctement reconstitués. La porte se déverrouille.
INDICES SI BLOQUÉ
INDICE 1
Chercher dans la zone autour de la borne des documents, affiches ou notes indiquant les achats des clients.
INDICE 2
Les partenaires sont souvent indiqués en bas ou en marge des fiches clients — ne pas se concentrer uniquement sur les articles principaux.
INDICE 3
Si la borne ne valide pas, vérifier si un partenaire a été omis ou si une quantité est incorrecte.
✓ VALIDATION Borne acceptée → Porte boutique déverrouillée → Passer à la Phase 02.
02
// INTÉRIEUR BOUTIQUE — OBJECTIF SECONDAIRE
OUVERTURE DU COFFRE-FORT
Ranger le contenu du coffre pour trouver la note cachée au fond contenant le mot de passe du PC de Zoé.
⚡ REQUIS
ℹ NOTE OPÉRATION Le coffre-fort est situé à l'intérieur de la boutique. Il est déjà ouvert ou accessible — l'enjeu n'est pas de le cracker mais de vider et ranger son contenu méthodiquement.
PROCÉDURE
  • 01.Localiser le coffre-fort dans la boutique.
  • 02.Sortir tout le contenu du coffre et le ranger de manière organisée à l'extérieur.
  • 03.Une fois le coffre entièrement vidé, inspecter le fond du coffre.
  • 04.Récupérer la petite note cachée au fond — elle contient le MDP du PC de Zoé.
  • 05.Mémoriser ou noter le mot de passe — il sera utilisé plus tard en Phase 04.
// La note peut être collée, glissée sous un fond amovible, ou placée dans un compartiment secret au fond du coffre. Inspecter minutieusement chaque surface intérieure.
✓ VALIDATION Note récupérée → MDP de Zoé obtenu → Nécessaire pour Phase 04.
03
// LS CUSTOM — RÉCUPÉRATION D'OUTIL
RÉCUPÉRER LA SCIE ÉLECTRIQUE
Se rendre dans le garage LS Custom et localiser la scie électrique — indispensable pour la Phase 07.
⟳ EN PARALLÈLE avec Phase 04 et 05
Cette phase peut être effectuée simultanément avec les Phases 04 et 05 si l'équipe se sépare.
LOCALISATION DE L'OUTIL
  • 01.Entrer dans le garage LS Custom.
  • 02.Se diriger vers le fond du garage, côté droite.
  • 03.Localiser la table de travail au fond à droite.
  • 04.Récupérer la scie électrique posée sur cette table.
  • 05.Conserver la scie — elle sera utilisée pour casser la porte en Phase 07.
📍 POSITION Fond du garage — Côté DROIT — Sur une table
✓ UTILISATION Phase 07 — Casser la porte de la zone finale
04
// LS CUSTOM — ACCÈS SYSTÈME
PC DE ZOÉ + NUMÉROS VIN
Utiliser le MDP récupéré en Phase 02 pour accéder au PC de Zoé, résoudre les énigmes et obtenir l'accès aux 4 voitures pour relever leurs VIN.
⟳ EN PARALLÈLE avec Phase 03 et 05
Cette phase nécessite le MDP trouvé en Phase 02 (note dans le coffre de Zoé).
ACCÈS AU PC
  • 01.Localiser le PC de Zoé dans LS Custom.
  • 02.Entrer le mot de passe trouvé dans le coffre-fort (Phase 02).
  • 03.Résoudre les énigmes présentes sur le PC pour débloquer l'accès aux données véhicules.
  • 04.Une fois les énigmes résolues, le système révèle les informations sur les 4 voitures ciblées.
RELEVÉ DES NUMÉROS VIN
  • 05.Localiser les 4 voitures dans le garage LS Custom selon les données du PC.
  • 06.Sur chaque voiture, trouver et noter le numéro VIN — généralement visible sur le tableau de bord côté conducteur ou sur le montant de porte.
  • 07.Relever les 4 VIN complets. Ces numéros seront effacés en Phase 10 pour finaliser l'opération.
⚠ CRITIQUE Ne pas effacer les VIN maintenant — cela sera fait en Phase 10 dans la zone finale, en parallèle avec l'ouverture de porte.
05
// LS CUSTOM — DÉCHIFFREMENT
DÉCODEUR DE COULEURS
Trouver le décodeur de couleurs caché derrière des conteneurs. Il fournit 9 couleurs avec leurs numéros correspondants — clé pour ouvrir l'arrière boutique.
⟳ EN PARALLÈLE avec Phase 03 et 04
LOCALISATION DU DÉCODEUR
  • 01.Explorer la zone LS Custom et localiser des conteneurs/containers.
  • 02.Chercher derrière ces conteneurs — le décodeur est caché là, pas visible de face.
  • 03.Récupérer le décodeur de couleurs — il liste 9 couleurs, chacune associée à un numéro.
UTILISATION DU DÉCODEUR
  • 04.Revenir vers les 4 voitures identifiées en Phase 04.
  • 05.Observer la couleur de chaque voiture et trouver le numéro correspondant dans le décodeur.
  • 06.Composer le code à partir des numéros de couleurs des 4 voitures — ce code ouvre l'arrière boutique cachée du LS Custom.
  • 07.Entrer le code sur la serrure ou le panneau de l'arrière boutique cachée.
ℹ MÉMO Le décodeur = 9 couleurs → 9 numéros. Chaque voiture a une couleur → un chiffre. Le code final est formé par les chiffres des 4 voitures dans l'ordre.
06
// ARRIÈRE BOUTIQUE LS CUSTOM
ZONE CACHÉE — LECTURE DU MUR
Dans l'arrière boutique cachée du LS, déchiffrer le code mural pour accéder à la phase finale.
↗ DÉBLOQUÉ par Phase 05
PROCÉDURE DANS LA ZONE CACHÉE
  • 01.Entrer dans la zone cachée du LS Custom (débloquée avec le code couleurs de la Phase 05).
  • 02.Examiner le mur dans cette zone — il y a deux faces à observer.
  • 03.Sur un côté du mur : l'ordre des éléments.
  • 04.Directement en face (mur opposé ou autre côté) : le nombre correspondant à chaque élément dans l'ordre.
  • 05.Combiner les deux informations pour composer le code final qui donnera accès à la phase finale (Phase 07).
MUR GAUCHE
→ L'ordre des éléments
→ Séquence à suivre
→ Lire de haut en bas
MUR FACE
→ Le nombre correspondant
→ Associer à l'ordre du 1er mur
→ Former le code
✓ VALIDATION Code mural déchiffré → Utiliser ce code pour entrer dans la Phase 07 (zone finale).
07
// ZONE FINALE — POINT D'ENTRÉE
ACCÈS À LA ZONE FINALE
Utiliser le code du mur et la scie électrique pour pénétrer dans la zone finale.
⚠ ZONE FINALE
ENTRÉE DANS LA ZONE FINALE
  • 01.Utiliser le code du mur obtenu en Phase 06 pour atteindre l'accès à la zone finale.
  • 02.Une fois devant la porte de la zone finale, utiliser la scie électrique récupérée en Phase 03 pour casser la porte.
  • 03.Entrer dans la zone finale.
🔧 REQUIS Scie électrique (Phase 03) indispensable.
🔑 REQUIS Code mural (Phase 06) pour localiser l'accès.
✓ VALIDATION Porte cassée → Zone finale accessible → Commencer Phases 08 et 10 en parallèle.
08
// ZONE FINALE — FOUILLE
TROUVER LA CLÉ ET LE BUREAU
Fouiller les établis de la zone finale pour trouver la clé perdue permettant d'accéder au bureau.
⟳ EN PARALLÈLE avec Phase 10
RECHERCHE DE LA CLÉ
  • 01.Entrer dans la zone finale (après Phase 07).
  • 02.Localiser les établis présents dans cette zone.
  • 03.Fouiller chaque établi méthodiquement — la clé est cachée dans l'un d'eux. Inspecter dessus, dessous, dans les tiroirs et compartiments.
  • 04.Récupérer la clé perdue.
  • 05.Utiliser la clé pour ouvrir le bureau dans cette zone.
ℹ STRATÉGIE Diviser l'équipe — une partie cherche la clé dans les établis, l'autre part effacer les VIN (Phase 10) pour optimiser le temps.
✓ VALIDATION Clé trouvée → Bureau ouvert → Accéder au bout de papier en Phase 09.
09
// BUREAU — DÉCHIFFREMENT FINAL
CODE DU BUREAU — 10 CODES À 4 CHIFFRES
Dans le bureau se trouve un bout de papier avec 10 codes à 4 chiffres. Un seul est le bon — il ouvre la porte vers l'étage en dessus.
↗ DÉBLOQUÉ par Phase 08
PROCÉDURE
  • 01.Entrer dans le bureau avec la clé obtenue en Phase 08.
  • 02.Localiser le bout de papier dans le bureau.
  • 03.Observer les 10 codes à 4 chiffres inscrits dessus.
  • 04.Chercher dans le bureau ou la zone des indices permettant d'identifier le bon code parmi les 10.
  • 05.Entrer le code correct sur la serrure de la porte menant à l'étage en dessous.
// 10 codes → 1 seul valide. Chercher des indices dans le bureau qui pointent vers le bon code : marques, surlignage, symboles, cohérence avec d'autres éléments trouvés dans l'escape.
⚠ MÉTHODE Ne pas tester tous les codes au hasard. Chercher d'abord un indice qui identifie lequel des 10 est le correct. Un élément du bureau ou de la zone devrait indiquer le bon.
✓ VALIDATION Bon code trouvé → Porte vers l'étage en dessous ouverte.
10
// ZONE FINALE — MISSION PRINCIPALE
EFFACER LES 4 NUMÉROS VIN
Localiser les 4 voitures dans la zone finale et effacer leurs numéros VIN — condition sine qua non pour accéder à l'extraction.
⟳ EN PARALLÈLE avec Phase 08 et 09
Peut être réalisé en simultané avec la recherche de la clé (Phase 08) et la résolution du code (Phase 09).
PROCÉDURE D'EFFACEMENT
  • 01.Localiser les 4 voitures dans la zone finale.
  • 02.Sur chaque voiture, localiser le numéro VIN (tableau de bord, montant de porte A, ou châssis).
  • 03.Effacer / supprimer / retirer le VIN sur la voiture 1.
  • 04.Effacer le VIN sur la voiture 2.
  • 05.Effacer le VIN sur la voiture 3.
  • 06.Effacer le VIN sur la voiture 4.
  • 07.Vérifier que les 4 VIN sont bien effacés — un seul oublié et l'accès à l'ascenseur reste bloqué.
CHECKLIST VIN
  • VIN Voiture 1 — effacé
  • VIN Voiture 2 — effacé
  • VIN Voiture 3 — effacé
  • VIN Voiture 4 — effacé
✓ VALIDATION 4 VIN effacés + Code bureau validé (Phase 09) → Prendre l'ascenseur → Phase 11.
